# Mathematical Visualization using GeoGebra

## Overview
This project explores mathematical visualization of geometric concepts using GeoGebra to enhance conceptual understanding and verify theoretical results.

## Objectives
- To visualize geometric properties such as tangents and cyclic quadrilaterals
- To bridge theoretical mathematics with computational tools
- To improve conceptual clarity through interactive models

## Topics Covered
- Tangents to Circles
- Cyclic Quadrilaterals
- Geometric Theorems and Properties

## Tools & Technologies
- GeoGebra
- LaTeX

## Methodology
Interactive models were created in GeoGebra to simulate and verify mathematical properties. Theoretical explanations and supporting arguments were documented using LaTeX.

## Outcome
Successfully developed multiple models that demonstrate mathematical concepts visually and support theoretical understanding.
